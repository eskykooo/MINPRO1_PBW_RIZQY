# Portfolio Website - Rizqy

Website portfolio pribadi yang menampilkan profil, pengalaman, sertifikat, dan showcase project video.  
Dibangun menggunakan HTML5, CSS3, Bootstrap 5, dan JavaScript dengan optimasi performa modern.

## Deskripsi Project

Website ini dibuat sebagai media personal branding untuk menampilkan kemampuan di bidang creative editing dan digital design.  
Menampilkan:

- Hero Section (Perkenalan)
- About Me
- Skills Progress Bar
- Certificates
- Video Project Showcase
- Responsive Navbar
- Optimasi Lazy Loading

### Nilai Tambah
Bootstrap

# Penjelasan Setiap Section

## Navbar

**Fungsi:**
Navigasi utama untuk berpindah antar section menggunakan anchor link.

**Teknologi:**
- Bootstrap 5 Navbar
- Responsive Collapse Menu
- Fixed Top Navigation

**Fitur:**
- Responsive di mobile
- Smooth scrolling
- Dark theme

## Hero Section

**Fungsi:**
Menampilkan foto profil dan deskripsi singkat.

**Fitur:**
- Fullscreen (100vh)
- Gradient background
- Rounded profile image
- Lazy loading image
- Call-to-action button

## About Section

**Fungsi:**
Menampilkan deskripsi diri, pengalaman, dan skill.

**Fitur:**
- Grid layout Bootstrap (2 kolom)
- Progress bar skill custom
- Responsive design

**Skill yang Ditampilkan:**
- After Effect (90%)
- Premier Pro (80%)
- Photoshop (95%)

## Certificates Section

**Fungsi:**
Menampilkan sertifikat dalam bentuk card.

**Fitur:**
- Bootstrap Card Component
- Shadow effect
- Responsive 3 column layout
- Lazy loading image

## Project Video Showcase

**Fungsi:**
Menampilkan hasil video project.

**Teknologi:**
- HTML5 `<video>`
- JavaScript Intersection Observer
- Lazy loading system

**Fitur:**
- Auto play saat terlihat di viewport
- Hover untuk unmute
- Performance optimized
- Card hover animation

## Footer

**Fungsi:**
Menampilkan informasi copyright.
